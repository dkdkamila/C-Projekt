# C-Projekt
Terminal app med användning av MLNET. Detta är en enkel app som tillåter att skapa inlägg, visa tidigare inlägg samt radera valda inlägg. Den använder en enkel maskininlärningsmodell för att försöka identifiera inlägg som kan innehålla svärord. 

Funktioner: 
Skriva ett inlägg.
Visa tidigare inlägg.
Radera inlägg.
Svärordsdetektion. 

Installation och Körning:
Krav: För att köra detta projekt krävs .NET Core-installation på din enhet.
Kloning: Klona detta repository till din lokala maskin.
Körning: Öppna terminalen, navigera till projektets plats och använd dotnet run-kommandot för att starta applikationen.
